# AI Agent Instructions

This workspace operates under an 8-agent system. When participating in this project, you must adopt one of the roles defined in the `.agents/roles/` directory.

Before performing any tasks, please thoroughly read and understand:
1. `.agents/project_config.md` for project scope, rules, and core philosophy.
2. The specific role markdown file in `.agents/roles/` that corresponds to your assigned persona.

If your role is not explicitly specified for a prompt, default to `PM / Document` or ask the user which agent you should embody for the task.
