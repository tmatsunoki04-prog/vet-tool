Privacy Policy

Last updated: March 6, 2026

1. Information We Do Not Collect

  • No Input Saving: Raw user input, missing-information answers, and generated prompts are not stored in our database or analytics logs.
  • Browser Memory: Some input may be temporarily held in browser memory during the active session to support regenerate behavior, but is not persisted by us to database or analytics logs.
  • No Accounts: This tool operates entirely anonymously. There is no user account or login system.

2. Information We Do Collect (Anonymous Statistics)

We collect limited anonymous analytics, including feature usage counts and AI-generated intent categories such as general topic classification. We do not store raw user input, generated prompts, or personal information.

To understand how our tool is used and to improve it, we collect completely anonymous usage statistics. This data cannot be traced back to you:

  • Event Types: We log when an action occurs (e.g., "generate", "copy", "share", "regenerate").
  • Abstract Intents: We classify usage into broad, non-specific category identifiers (e.g., intent_category: "business", intent_subtype: "finance", intent_action: "write_report").
  • Input Length Buckets: We log the rough size of the request (e.g., "small", "medium", "large") to optimize performance.
  • Timestamps: We record the time an event occurred.

3. Third-Party Services

We may use basic hosting and analytics providers to process the anonymous data mentioned above. User input may be sent to Google Gemini strictly for prompt generation/regeneration. These providers are bound by strict data processing agreements and do not train models on API input.

4. Changes to this Policy

We may update our Privacy Policy occasionally. We will notify you of any changes by posting the new Privacy Policy on this page.
